📋 개요 & 배경

문제 인식

오늘날 기업과 구성원은 쏟아지는 뉴스·리포트·트렌드 속에서 "지금 우리가 주목해야 할 변화는 무엇인가?"를 파악하는 데 많은 시간과 에너지를 소비합니다.

  • 하루에도 수백 건의 산업 뉴스가 발생하지만, 전략적으로 중요한 신호를 골라내는 작업은 여전히 사람이 수동으로 수행합니다.
  • 감지된 변화 신호를 "우리 조직의 맥락"에 맞게 해석하고 Action Item으로 전환하는 데 전문성과 시간이 필요합니다.
  • 조직·직군·직종 각 레벨마다 필요한 인사이트 포맷이 달라, 단일 리포트로는 조직 전체를 커버하기 어렵습니다.

솔루션 개요 - 3단계 파이프라인

①
수집 (Collect) - 사용자가 정의한 키워드 + 도메인 기반으로 뉴스·리포트 자동 크롤링
→ Input: 키워드, 직군 (영업/R&D/생산/경영지원/IT 등), 직종 (기획·인사·개발·영업 등), 정기 구독 여부, 이메일
②
탐지 (Detect) - 변화 신호 분류, 중요도 스코어링, 트렌드 패턴 추출
→ System Prompt 명확하게 작성
③
가이드 (Guide) — 직군·직종에 맞춘 전략 액션 제안

👤 타겟

페르소나직군 / 직종니즈사용 방식
🏢 전략 담당자 경영지원직군 / 전략기획 경쟁사·기술 트렌드를 빠르게 파악해 경영진 보고에 활용 주간 브리핑 구독 + 챗봇 질의
🔬 기술 리더 IT직군 / 백엔드 개발자 AI·자동화 등 기술 변화가 우리 시스템에 미치는 영향 분석 기술 키워드 맞춤 설정 + 일일 알림
📊 세일즈/BD 영업직군 / 서비스 기획 고객 산업 변화 신호를 포착해 영업 기회 발굴 고객사 키워드 모니터링 + 브리핑
👷 실무 담당자 생산직군 / 영업기획 프로젝트와 관련된 시장 변화를 놓치지 않고 대응 프로젝트별 키워드 그룹 설정
🏢
1차 타겟: 코코아소프트 내부 - AI팀, RPA팀, 영상제작팀
2차 타겟 (사업화): 업종·직군에 관계없이, 변화에 앞서 대응하고 싶은 모든 직장인

⚙️ 핵심 기능

기능 1 · 키워드 & 도메인 설정 (온보딩)

  • 사용자가 모니터링할 키워드를 자유 입력 (예: "생성형 AI", "RPA 시장", "자동화 트렌드")
  • 직군 (산업 도메인) 선택 (IT/금융/제조/유통 등) 으로 신호 필터링 정밀도 향상
  • 직종 설정 → 더 상세한 필터링 및 범위를 좁혀서 정밀도 향상
  • 알림 채널 연결: 이메일

기능 2 · 뉴스 & 신호 자동 수집 (n8n 워크플로우)

  • n8n 스케줄러: 매일 06:00 자동 실행 (정기 메일 발송 선택 시)
  • 수집 소스: RSS 피드 (네이버 뉴스·구글 뉴스), 공개 API, 산업 리포트 사이트 등
  • 키워드 매칭 필터링 → 관련 기사만 추출
  • 중복 제거 + 신선도 스코어링 (최신 24시간 가중치 부여)

기능 3 · AI 변화 신호 탐지 & 분석 (Claude + Dify)

  • 신호 분류: 기술 변화 / 시장 변화 / 규제 변화 / 경쟁 변화 4가지 카테고리
  • 중요도 스코어링: 0–10점 자동 부여 (조직 관련성 × 변화 속도 × 영향 범위)
  • 트렌드 패턴 감지: 동일 주제 기사가 급증하는 "신호 급등" 알림

기능 4 · 직종별 전략 가이드 생성

🔍
키워드 "AI" 검색 시 직군·직종별 가이드 예시
🏢 경영지원직군 / 전략기획: AI 도입 관련 정책·규제 변화 신호 + 경영진 의사결정을 위한 AI 투자 우선순위 포인트 + 조직 리스크 대응 방향
💻 IT직군 / 백엔드 개발자: AI 관련 기술 트렌드 신호 (LLM, RAG, Agent 등) + 관련 기술 레퍼런스 링크 + 당장 적용 가능한 학습·실습 To-Do
📣 영업직군 / 서비스 기획: AI 기반 서비스 시장 변화 신호 + 고객 니즈 변화에 따른 서비스 방향성 제안 + 단기 Action Item
🏭 생산직군 / 영업기획: AI 자동화로 인한 산업 구조 변화 신호 + 영업 전략 조정 포인트 + 실행 우선순위
